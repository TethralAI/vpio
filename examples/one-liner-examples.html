Method 1: Zero JavaScript - HTML Data Attributes

Just add data attributes to any button. VibePay SDK auto-detects and converts it.

<!-- Step 1: Include SDK -->
<script src="vibepay-sdk.js"></script>

<!-- Step 2: Add data-vibepay to any button -->
<button data-vibepay="10" data-description="Premium Plan">Subscribe</button>

üëÜ This button was auto-converted by VibePay SDK

Method 2: CSS Class - Add .vibepay-button

Add the CSS class and data-amount. VibePay handles the rest.

<button class="vibepay-button" data-amount="25" data-description="Pro Plan"> Get Pro Plan </button>

üëÜ Auto-styled and functional via CSS class

Method 3: JavaScript One-Liner - VibePay.payButton()

One line of JavaScript to convert any element into a payment button.

// Turn any button into a payment button
VibePay.payButton('#my-btn', 50, 'Donation');

üëÜ Click "Convert" to see the magic

Method 4: Auto-Inject - VibePay.injectButton()

Automatically inject a payment button into any container.

// Inject a payment button anywhere
VibePay.injectButton('#container', 100, 'Enterprise Plan');

Method 5: Programmatic - VibePay.quickPay()

Trigger payments from anywhere in your code with one function call.

// Trigger payment from anywhere
VibePay.quickPay(75, 'Custom Item').then(result => console.log(result));

üëÜ Direct payment trigger (check console for result)
